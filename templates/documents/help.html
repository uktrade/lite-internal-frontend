Help with creating a document

Formatting

To put a title in your template, use a hash:

# This is a title

To make bullet points, use asterisks:

* point 1
* point 2
* point 3

To add a horizontal line, use three dashes:

First paragraph
---
Second paragraph

Personalisation

Use double brackets to personalise your document:

{% verbatim %}Hello {% first name %}, your reference is {% refence_number %}{% endverbatim %}
